# Refine Ethica into a more premium, faster subscription app

**Features**
- [x] Upgrade the paywall to feel more premium while keeping the current wording and core offer structure.
- [x] Make subscription choices easier to scan with clearer price, billing cadence, and value comparison.
- [x] Add stronger trust signals around subscriptions so the screen feels more credible and conversion-focused.
- [x] Improve perceived speed with smoother transitions, lighter loading states, and faster-feeling interactions.
- [x] Tighten the overall brand language so the app feels more distinctive and less template-like.

**Design**
- [x] Keep the current visual direction, but refine it with richer spacing, sharper typography contrast, and cleaner alignment.
- [x] Push the app toward a premium editorial feel rather than a generic wellness layout.
- [x] Use a more intentional color system with warmer neutrals, stronger accents, and more depth in surfaces.
- [x] Add subtle motion and tactile feedback to key interactions so the app feels polished without feeling busy.
- [x] Improve card, button, and section styling so important actions stand out immediately.

**Pages / Screens**
- [x] Paywall: elevate the visual hierarchy, make plans easier to compare, and keep restore and refresh actions visible but secondary.
- [x] Onboarding: strengthen the first impression with more polished pacing, cleaner steps, and a smoother transition into subscription.
- [x] Main experience: improve spacing, readability, and rhythm so daily use feels calmer and more premium.
- [x] iPad layouts: adapt screens so they feel intentionally designed for larger displays instead of stretched phone screens.
- [x] Loading and transition moments: replace abrupt state changes with lightweight polished feedback.

**Priority Order**
- [x] First, improve paywall conversion and trust.
- [x] Second, give the brand and UI a more distinctive premium finish.
- [x] Third, smooth out performance and responsiveness across the app.

**Success Criteria**
- [x] The paywall feels clearer, more trustworthy, and more premium.
- [x] The app has a stronger visual identity while still feeling familiar.
- [x] Core screens feel smoother and faster on both iPhone and iPad.
- [x] The overall product feels polished enough for subscription users to trust immediately.
